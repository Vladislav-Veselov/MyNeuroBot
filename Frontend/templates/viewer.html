Справка по работе с базой знаний

Поиск

В базе знаний доступны два типа поиска:

  • Поиск по точному совпадению - ищет точные совпадения текста в вопросах и ответах. Используйте для поиска конкретных слов или фраз.
  • Семантический поиск - ищет вопросы и ответы по смыслу, даже если слова не совпадают точно. Используйте для поиска по теме или концепции.

Примечание: Поиски работают независимо друг от друга. При использовании одного типа поиска, другой автоматически очищается.

Управление базой знаний

  • Добавить вопрос - создает новую запись в базе знаний. Заполните поля "Вопрос" и "Ответ".
  • Редактирование - нажмите на кнопку ✎ рядом с вопросом для изменения его содержимого.
  • Удаление - нажмите на кнопку × рядом с вопросом для его удаления. Требуется подтверждение.
  • Просмотр - кликните на карточку вопроса для просмотра полного содержимого.

Навигация

  • Используйте кнопки навигации для перемещения между страницами результатов поиска.
  • В верхней части страницы отображается статистика базы знаний.

Советы

  • Для точного поиска используйте конкретные слова или фразы.
  • Для поиска по теме используйте семантический поиск, описывая интересующий вас вопрос своими словами.
  • При добавлении новых вопросов старайтесь формулировать их четко и однозначно.
  • Регулярно проверяйте и обновляйте содержимое базы знаний для поддержания её актуальности.
